# Online library

Сайт является электронной библиотекой.

## Как запустить

Для запуска библиотеки у вас уже должен быть установлен Python 3.

- Скачайте код
- Установите зависимости командой `pip install -r requirements.txt`
- Запустите сервер командой `python render_website.py`

После этого переходите по ссылке [http://127.0.0.1:5500/pages/index1.html], вы увидите первую страницу.

Пример, как выглядит библиотека доступен по следующей ссылке [https://alinarogova123-arch.github.io/web_layout_03/pages/index1.html]

## Цели проекта

Код написан в учебных целях — это урок в курсе по Python и веб-разработке на сайте [Devman](https://dvmn.org).
